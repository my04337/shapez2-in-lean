# `cbv` — コール・バイ・バリュー簡約を実行する

**カテゴリ**: 簡約 | **定義元**: `Lean.Parser.Tactic.cbv` | **ソース**: Lean4 core

---

## 概要

`cbv` はコール・バイ・バリュー（CBV）戦略でゴールを簡約するタクティクである。引数を先に評価してから関数本体に代入する戦略で、`whnf` と異なり内部まで完全に簡約する。`reduce` に似ているが、CBV 戦略に基づく。

---

## ゴールパターン

**適用前**
```lean
⊢ (fun x => x + 1) 5 = 6
```

**適用後**
```lean
⊢ 6 = 6
```

---

## 構文

```lean
cbv
```

---

## 必須 import

Lean4 core 組み込みのため不要。

---

## オプション一覧

特になし。

---

## Pros（使うべき場面）

- 完全な CBV 評価が必要な場合
- `reduce` の代替として使用可能
- core 組み込みで import 不要

---

## Cons（注意・失敗ケース）

- 大きな式では非常に遅くなる場合がある
- ゴールが巨大に展開されるリスク
- `norm_num` や `decide` の方が適切なことが多い

---

## コードサンプル

### サンプル 1: 数値計算

```lean
-- cbv は数値を簡約して自動的にゴールを閉じる
example : 2 + 3 = 5 := by
  cbv
```
