# `trace` — トレースメッセージを表示する

**カテゴリ**: デバッグ | **定義元**: `Lean.Elab.Tactic.trace` | **ソース**: Lean4 core

---

## 概要

`trace "{msg}"` は証明中にデバッグ用のメッセージを Lean のメッセージウィンドウに出力する。ゴールや仮定の状態を途中で確認したい場合、変数の値を表示したい場合に使う。証明の進行には影響しない。

---

## ゴールパターン

任意のゴールに対して使用可能。ゴールを変更しない。

---

## 構文

```lean
trace "{msg}"
trace "x = {x}"     -- 補間あり
```

---

## 必須 import

Lean4 core 組み込みのため不要。

---

## オプション一覧

| 構文 | 意味 |
|---|---|
| `trace "msg"` | 文字列メッセージを出力 |
| `trace "x = {x}"` | 変数の値を補間して出力 |

---

## Pros（使うべき場面）

- 証明の中間状態のデバッグ
- 変数の値を確認
- タクティクの実行順序を追跡

---

## Cons（注意・失敗ケース）

- デバッグ専用で証明を進めない
- 最終的な証明からは除去すべき
- 大量の出力でメッセージが読みにくくなる場合がある

---

## コードサンプル

### サンプル 1: 基本メッセージ

```lean
example : True := by
  trace "ここを通過"
  trivial
```

### サンプル 2: 変数の補間

```lean
example : ∀ n : Nat, n = n := by
  intro n
  trace "n の型は Nat"
  rfl
```
