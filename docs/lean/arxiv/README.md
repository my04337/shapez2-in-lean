# arXiv 論文リファレンス

Lean 4 / 定理証明に関連する arXiv 論文の要約・分析を格納するディレクトリです。

## 目的

- プロジェクトに関連する学術論文の要約と洞察をプロジェクト内で参照可能にする
- 精読を支援する目次・逆引きインデックスを提供する
- 出典とライセンスを明確に管理する

## ディレクトリ構造

各論文は `arXiv番号-短縮タイトル/` のサブディレクトリに格納されます。

```
docs/lean/arxiv/
├── README.md                                    # 本ファイル
└── 2603.24465-mechanic-sorrifier/               # 各論文ディレクトリ
    ├── README.md                                # 出典・ライセンス
    └── summary.md                               # 要約・分析・目次
```

## 利用上の注意

- 各論文ディレクトリの `README.md` に元論文のライセンスと帰属表示義務が記載されています。利用前に必ず確認してください
- これらのファイルは論文の**要約・分析**であり、原文のコピーではありません
- プロジェクト固有のナレッジは `docs/s2il/` に記録してください
