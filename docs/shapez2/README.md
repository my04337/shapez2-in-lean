# Shapez2 ゲームドメイン仕様

Shapez2 のゲーム内仕様・用語・操作セマンティクスをまとめたドキュメント群の入口ページです。

ゲーム仕様の確認・実装・リファクタリング時にはまずここを参照すること。

---

## ファイル一覧

### 基礎用語・概念

| ファイル | 概要 |
|---|---|
| [glossary.md](glossary.md) | ゲーム内用語集。シェイプ・建物・加工操作などをナレッジパネルのカテゴリ別に定義 |
| [adjacency.md](adjacency.md) | 隣接関係の定義。結晶結合判定・回転挙動・積み重ね落下判定など、複数操作の基盤となる共通概念 |

### 加工操作の仕様

| ファイル | 概要 |
|---|---|
| [crystal-shatter.md](crystal-shatter.md) | 結晶の砕け散り操作の厳密な仕様。脆弱な結晶の破壊と結合クラスタへの連鎖を定義 |
| [falling.md](falling.md) | 落下操作の厳密な仕様。支えのないシェイプの塊が下方に移動する物理挙動を定義 |

### 高レベル概念

| ファイル | 概要 |
|---|---|
| [mam.md](mam.md) | MaM（Make Anything Machine）の形式化。任意のシェイプを自動生産する設備の概念と動作原理 |

## エージェント向け参照ガイダンス

### いつ参照するか

| 作業 | 参照先 |
|---|---|
| 新しい操作の Lean 実装着手前 | まず `glossary.md` で用語を確認し、対応する仕様ファイルを読む |
| 隣接関係・結合判定に関わる実装 | `adjacency.md` を必ず参照 |
| 結晶の砕け散りに関わる実装・証明 | `crystal-shatter.md` + `adjacency.md` |
| 落下処理に関わる実装・証明 | `falling.md` + `adjacency.md` |
| ゲーム仕様の曖昧な点に遭遇した場合 | `glossary.md` → 対応する仕様ファイルの順に確認 |

### 新規ファイルの追加基準

- **追加する**: 新しい操作（Painter、Stacker 等）の厳密な仕様を文書化する場合
- **追加しない**: 実装の詳細（Lean コードの具体的な設計判断）は `docs/knowledge/` に記録する
- **ファイル名**: 操作名をケバブケースで（例: `pin-pusher.md`）
- 新規ファイルを追加した際は、この README のファイル一覧にも追記すること
