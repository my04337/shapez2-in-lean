# 隣接関係 (Adjacency)

Shapez2 におけるシェイプ内の **隣接関係 (Adjacency)** の定義。

隣接関係は結晶の結合判定、回転の挙動理解、積み重ね時の落下判定など、
複数の加工操作の基盤となる共通概念である。

---

## 1. 概要

シェイプは **レイヤ (Layer)** の積み重ねで構成され、各レイヤは **4つの象限 (Quarter)** を持つ。
隣接関係には以下の2種類がある:

| 種類 | スコープ | 条件 |
|---|---|---|
| **同レイヤ内隣接** | 同一レイヤ内の2象限 | 方角が円環上で隣り合う |
| **上下レイヤ間隣接（垂直隣接）** | 異なるレイヤの2象限 | 同方角かつレイヤが直上/直下 |

---

## 2. 方角 (Direction)

各レイヤは4つの方角に対応する象限を持つ。

| 方角 | 略称 | 位置 |
|---|---|---|
| Northeast | NE | 右上 |
| Southeast | SE | 右下 |
| Southwest | SW | 左下 |
| Northwest | NW | 左上 |

### 東西の半分 (East / West Half)

シェイプを南北方向の切断線で二分すると、東側と西側に分かれる。

| 半分 | 含まれる方角 |
|---|---|
| **East Half** (東側) | NE, SE |
| **West Half** (西側) | NW, SW |

この分類は切断操作（切断処理機・切断機・スワップ機）で使用される。

### 180° 回転との関係

180° 回転は NE↔SW、SE↔NW の入れ替えとして作用する。
この結果、East Half と West Half が入れ替わる:

| 回転前 | 回転後 |
|---|---|
| NE (East) | SW (West) |
| SE (East) | NW (West) |
| SW (West) | NE (East) |
| NW (West) | SE (East) |

---

## 3. 同レイヤ内の隣接関係

同一レイヤ内の4象限を円環状に並べた隣り合いとして定義される。

```
    NW ── NE
    │      │
    SW ── SE
```

### 隣接ペア

以下の4組が隣接ペアである:

| # | ペア | 備考 |
|---|---|---|
| 1 | NE ↔ SE | 東側の縦隣接 |
| 2 | SE ↔ SW | 南側の横隣接 |
| 3 | SW ↔ NW | 西側の縦隣接 |
| 4 | NW ↔ NE | 北側の横隣接 |

> **注意**: NE ↔ SW や SE ↔ NW のような **対角ペアは隣接しない**。

### 隣接関係の性質

| 性質 | 説明 |
|---|---|
| **対称性** | A と B が隣接 ⟺ B と A が隣接 |
| **非反射性** | どの象限も自身とは隣接しない |
| **回転保存** | 180° 回転（NE↔SW, SE↔NW の入れ替え）後も隣接関係は保存される |

### Lean での実装

```lean
def Direction.adjacent : Direction → Direction → Bool
    | ne, se | se, ne => true
    | se, sw | sw, se => true
    | sw, nw | nw, sw => true
    | nw, ne | ne, nw => true
    | _, _             => false
```

参照: `S2IL/Shape/QuarterPos.lean`

## 4. 上下レイヤ間の隣接関係（垂直隣接）

### レイヤインデックス

シェイプは最大4レイヤで構成され、下から順に番号が振られる。

| インデックス | 位置 |
|---|---|
| L1 | 最下層 |
| L2 | 下から2番目 |
| L3 | 下から3番目 |
| L4 | 最上層 |

### 垂直隣接ペア

直上・直下のレイヤ同士が垂直に隣接する。

| # | ペア | 備考 |
|---|---|---|
| 1 | L1 ↔ L2 | 最下層と2層目 |
| 2 | L2 ↔ L3 | 2層目と3層目 |
| 3 | L3 ↔ L4 | 3層目と最上層 |

> **注意**: L1 ↔ L3、L1 ↔ L4、L2 ↔ L4 のような **飛び越しペアは隣接しない**。

### 垂直隣接の性質

| 性質 | 説明 |
|---|---|
| **対称性** | L₁ と L₂ が垂直隣接 ⟺ L₂ と L₁ が垂直隣接 |
| **非反射性** | どのレイヤも自身とは垂直隣接しない |

### 上下レイヤ間の象限の隣接条件

レイヤ L_a の象限と レイヤ L_b の象限が上下レイヤ間で隣接する条件:

1. L_a と L_b が **垂直に隣接** するレイヤである
2. 両象限が **同じ方角** (NE, SE, SW, NW) に位置する

> **注意**: 異なる方角の象限同士（例: L1 の NE と L2 の SE）は上下レイヤ間で隣接しない。

### Lean での実装

```lean
def LayerIndex.verticallyAdjacent : LayerIndex → LayerIndex → Bool
    | l1, l2 | l2, l1 => true
    | l2, l3 | l3, l2 => true
    | l3, l4 | l4, l3 => true
    | _, _             => false
```

参照: `S2IL/Shape/QuarterPos.lean`

## 5. 象限位置 (QuarterPos)

シェイプ内の象限の絶対位置は、**レイヤインデックス** と **方角** の組で表される。

```lean
structure QuarterPos where
    layer : LayerIndex
    dir   : Direction
```

最大で 4レイヤ × 4方角 = **16個** の象限位置が存在する。
ただしシェイプのレイヤ数によって有効な位置数は異なる。

| レイヤ数 | 有効な象限位置数 |
|---|---|
| 1 | 4 |
| 2 | 8 |
| 3 | 12 |
| 4 | 16 |

参照: `S2IL/Shape/QuarterPos.lean`
