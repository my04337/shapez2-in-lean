# HOWTO

Shapez2 の加工操作に関する実践的な解説をまとめたドキュメントです。
用語の定義は [GLOSSARY.md](GLOSSARY.md) を参照してください。

---

## Vortex (渦) への納品

Vortex に納品する際、期待されるシェイプに対して **Point Symmetry (点対称)** な図形であれば、回転の向きに関わらず納品が受け付けられる。回転工程を省略できる場合があるため活用したい。

**Line Symmetry (線対称)** のみの図形は点対称ではないため、向きによっては納品が拒否されるので注意。

## Cutting & Rotating (切断と回転)

### 切断の向きと方角

すべての切断機はシェイプの **West Half (西側の半分)** を切断する。切断・スワップの位置は建物の設置方向やカメラ視点に依存せず、**Cardinal Directions (方角)** により常に一定に決定される。

シェイプを **Rotator (回転機)** で回転させることで、どちら側が West Half になるかを制御できる。

### 象限への分解

「切断 → 回転 → 切断」の操作により、シェイプを **Quarters (象限)** 単位に分解できる。

### 積み重ねへの応用

シェイプは上から下へ落下するように積み重ねられる。回転によってどのパーツがどこに落ちるかを制御できるため、2つの半分を1つの完全なシェイプに合わせたり、意図した位置に積み重ねたりすることができる。

## Stacking (積み重ね)

### 積層機の挙動

**Stacker (積層機)** は2つのフロアからシェイプを受け取り、上のシェイプを下のシェイプの上に落とす。切断されたパーツは固い面に当たるまで落下する。

両方のシェイプが完全な場合は互いの上に重なる。クォーターが空または欠落している場合、一部のパーツはさらに落下することがある。

### レイヤー制限

シェイプは最大 **4 レイヤー**（Insane 難易度では **5 レイヤー**）まで積み重ねられる。超過分のレイヤーは廃棄される。

---

## Pin Pusher (ピン押し機)

### サポートピンの役割

接続されていないクォーターは積み重ね時に落下する。**Pin Pusher (ピン押し機)** は第1レイヤーの空でない各象限の下に **Support Pin (サポートピン)** を押し込む。ピンはクォーターを保持し、それ自体も積み重ねられるため、通常では不可能な大きな隙間のある図形や複数レイヤーにわたる図形を製造できる。

### レイヤー制限の逆利用

Pin Pusher でシェイプを押し上げてレイヤー上限を超えると、超過分が廃棄される。この性質を逆利用することで、ピンを最上位に配置した図形を作ることもできる。

---

## Crystals (結晶)

### 結晶の生成

**Crystal Generator (結晶製造機)** は、供給された液剤をシェイプの隙間やピンに充填し、結晶として硬化させる。複数の結晶製造機を連続させることで、異なる色の結晶を1つの図形内に組み合わせることができる。

### 結晶の特性

結晶は **Fragile (脆弱)** であり、積み重ね時の落下で砕ける。切断すると **Shatter (砕け散り)** が発生し、同じ図形の同一・上下レイヤーでつながるすべての結晶が一斉に破壊される。

積み重ねによる破壊は避けられないが、切断しない限り **Swapper (スワップ機)** で結晶の半分を入れ替えることはできる。

### 応用：結晶下の隙間 (Hard / Insane 限定)

入れ替え中に意図的に **Break Crystals (結晶を壊す)** ことで、結晶の下に隙間のある図形を作ることができる。

## Wires (ワイヤー)

### 論理信号の種類

ワイヤー建物は **Logical Signals (論理信号)** を輸送・処理する。信号は次の5種類:

| 種類 | 内容 |
|---|---|
| Number (数値) | 整数値 (0, 1, 2, …) |
| Color (色) | r, g, b, c, m, y, w, u |
| Shape (図形) | 任意の図形 (例: `CuCuCuCu`) |
| Empty (空) | null とも呼ばれる |
| Conflict (コンフリクト) | 異なる信号を持つ2出力を接続したとき |

Empty と数値 0 は **Off** 、それ以外はすべて **On** に分類される。

### Belt Filter の動作

| 入力信号 | 動作 |
|---|---|
| Shape | 一致する図形は通過、不一致は横に除外 |
| Number (0) | 除外。0 以外は通過 |
| Empty / Conflict | ベルトを完全に遮断 |

### ランダムオペレーター図形と MaM

オペレーターレベルでは **Randomized Shape (ランダム図形)** 目標が導入される。**Operator Shape Receiver (オペレーター図形レシーバー)** で目標図形を論理信号として取得し、任意の図形を自動生産する **Make Anything Machine (MaM)** を構築することが推奨される。

### MaM の基本原理

**Simulated Buildings (シミュレーション設備)** を使うと、論理信号内の図形に対して切断・回転などの処理を仮想的に実行できる。切断処理機・回転機・切断処理機の順で処理を繰り返すことで、図形信号を指定象限のみに絞り込める。これを全象限 × 全図形タイプに対して行い結果を積み重ねることで、任意の無着色1レイヤー図形を製造できる。
